Remedial Dungeons and Dragons

View Project

Description: A chat website where you can create a profile and create/select topics and discuss several topics, all with the theme of Dungeons and Dragons
Class/Organization: ACIT-2520 Developing Web Applications
Date of Completion: March-6-2017


Reflection of Project:
This my midterm project for the Web Application Development course that was part of my CIT Diploma program. This project introduced a lot of new elements that we touched in during class, such as sessions, servers, sockets, and web packing. I am proud of this project as it was enjoyable to create and I got to express a creative freedom behind it by picking the theme. Also, it was actually used by my Dungeons and Dragons group to communicate for a few months after it was completed. I learned quite a few important methods and a brain set when developing a project from scratch. I would have thought of the whole picture before starting instead of starting with what I know (front-end development) and managing to make the rest work with what I have already created (back-end development). In my later projects, I laid out project goals, with ideas on how to complete these tasks before jumping into it. This planning prior to starting has given myself an easier time handling complex problems with projects.
